HTML — Structure of the Web

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It defines the meaning and structure of web content using elements represented by tags.

  • Semantic elements like <header>, <main>, <footer>
  • Forms, tables, media elements
  • Accessibility through ARIA attributes
  • HTML5 introduced Canvas, Audio, Video APIs

CSS — Style & Layout

CSS (Cascading Style Sheets) controls the visual presentation of HTML documents. Modern CSS enables powerful layouts, animations, and responsive designs without JavaScript.

  • Flexbox and CSS Grid for layouts
  • Custom Properties (CSS Variables)
  • Keyframe animations and transitions
  • Media queries for responsive design

JavaScript — Interactivity

JavaScript is the programming language of the web, enabling dynamic content, interactivity, and complex logic directly in the browser without page reloads.

  • Event-driven programming model
  • Promises and async/await for asynchronous code
  • ES6+ features: arrow functions, destructuring, modules
  • Full DOM manipulation capabilities

jQuery — Write Less, Do More

jQuery is a fast, small JavaScript library that simplifies HTML traversal, event handling, animation, and Ajax interactions for rapid web development.

  • Simple DOM selection with $(selector)
  • Built-in effects: fadeIn, slideUp, animate
  • Cross-browser compatible event handling
  • Easy Ajax with $.get() and $.post()

DOM — Document Object Model

The DOM is a programming interface for HTML documents. It represents the document as a tree of nodes, allowing scripts to dynamically access and update the content, structure, and styles.

  • Tree-based document representation
  • Node types: Element, Text, Comment
  • Event bubbling and capturing
  • MutationObserver for tracking DOM changes
